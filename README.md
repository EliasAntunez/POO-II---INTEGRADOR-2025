## Descripción del Proyecto

Este proyecto consiste en el desarrollo de un **sistema de facturación de servicios** para una empresa que gestiona **cuentas de clientes con diferentes condiciones fiscales** según la legislación argentina (IVA).
El sistema permite realizar la **gestión de clientes y cuentas**, **facturación masiva o individual por período**, **anulación de facturas** y **registro de pagos**.

El objetivo es aplicar los conceptos de **Programación Orientada a Objetos**, incluyendo diseño modular, encapsulamiento, herencia, polimorfismo y uso de patrones de diseño, siguiendo un enfoque iterativo e incremental.

## Funcionalidades principales

* Gestión de clientes y sus condiciones fiscales.
* Generación de facturas individuales y masivas.
* Anulación de facturas.
* Registro de pagos.
* Control de cuentas corrientes por cliente.
