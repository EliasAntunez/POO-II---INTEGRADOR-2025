# Retrospectiva Iteración 01

## Logros alcanzados

- Se creó la estructura base del proyecto utilizando Spring Boot y Maven.
- Se definieron los modelos principales: Cliente, Servicio y ClienteServicio, junto con sus respectivos repositorios y servicios.
- Se implementaron los controladores para la gestión de clientes, servicios y la relación entre ambos.
- Se desarrollaron las vistas principales para crear, listar y modificar clientes, servicios y clientes-servicios.
- Se configuró correctamente el entorno de desarrollo y el control de versiones con Git.
- Se logró la integración básica entre backend y frontend, permitiendo operaciones CRUD.

## Aspectos no cumplidos

- Falta implementar validaciones más robustas en los formularios y en el backend.
- No se realizó la integración con bases de datos externas, actualmente se utiliza almacenamiento en memoria.
- No se agregaron pruebas unitarias ni de integración suficientes.
- La documentación técnica está incompleta.

## Retos enfrentados

- Dificultad para definir la estructura inicial del proyecto y la organización de los paquetes.
- Problemas con la configuración de dependencias y el correcto funcionamiento de Maven.
- Falta de experiencia previa con Spring Boot en algunos integrantes del equipo.
- Coordinación para trabajar en paralelo sin generar conflictos en el repositorio.

## Plan de mejora para la próxima iteración

- Priorizar la implementación de validaciones y manejo de errores tanto en frontend como en backend.
- Agregar pruebas unitarias y de integración para asegurar la calidad del código.
- Mejorar la documentación técnica y funcional del proyecto.
- Organizar reuniones de seguimiento más frecuentes para mejorar la comunicación y coordinación del equipo.
- Definir claramente las tareas y responsables para evitar solapamientos y mejorar la eficiencia.
