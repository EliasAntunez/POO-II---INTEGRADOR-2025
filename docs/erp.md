# Requisitos funcionales:

## RF-01: Gestión de Clientes
**Descripción:** El sistema debe permitir gestionar los clientes (ABM de clientes), teniendo en cuenta su condición fiscal.

## RF-02: Gestión de Servicios
**Descripción:** El sistema debe permitir gestionar los servicios (ABM de servicios).
 
## RF-03: Facturación Masiva
**Descripción:** El sistema debe permitir emitir muchas facturas a la vez (por ejemplo, 100 clientes en un clic), en un periodo especifico (un mes).

## RF-04: Facturación Individual
**Descripción:** El sistema debe permitir emitir una factura de un cliente en especifico.

## RF-05: Anulacion de Facturas
**Descripción:** El sistema debe permitir anular facturas si fue emitida por error (importe incorrecto, servicio duplicado, etc.), generando una nota de crédito.

## RF-06: Registro de Pagos
**Descripción:** El sistema debe permitir registrar cuándo un cliente paga una factura.
